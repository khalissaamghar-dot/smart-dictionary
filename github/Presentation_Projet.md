# DOSSIER DE PRÉSENTATION : SMART SCIENTIFIC DICTIONARY

## 1. Résumé du Projet
Le **Smart Scientific Dictionary** est une plateforme éducative moderne conçue pour accompagner les élèves de lycée dans l'apprentissage des sciences. Elle combine la rigueur d'un dictionnaire technique avec la puissance de l'intelligence artificielle générative.

## 2. Les Enjeux
L'accès au savoir scientifique est souvent freiné par deux barrières :
* **La Barrière Linguistique :** De nombreux élèves maîtrisent mieux une langue que l'autre pour les concepts techniques.
* **La Barrière de la Complexité :** Les définitions standards sont souvent trop rigides ou abstraites.

## 3. Caractéristiques Principales
* **Système Multilingue (4 Langues) :** Support intégral du Français, Arabe, Anglais et Amazigh (Tifinagh).
* **Assistant Scientifique IA :** Un chatbot intégré basé sur le modèle Gemini de Google, capable d'expliquer les termes complexes de manière pédagogique.
* **Interface Premium :** Design "Glassmorphism" moderne, responsive et optimisé pour une lecture agréable sur mobile et ordinateur.
* **Base de Données Firestore :** Gestion dynamique des termes et des favoris pour chaque utilisateur.

## 4. Architecture Technique
Le projet repose sur une architecture Cloud performante :
* **Frontend :** HTML5, Vanilla CSS3, JavaScript (ES6+).
* **Backend :** Node.js via Vercel Serverless Functions.
* **Base de Données :** Google Firebase Firestore.
* **Intelligence Artificielle :** Google Gemini API (Modèle 1.5 Flash/Pro).
* **Déploiement :** Intégration continue (CI/CD) via GitHub et Vercel.

## 5. Perspectives d'Évolution
* Intégration de schémas interactifs en 3D.
* Mode hors-ligne (PWA).
* Quiz d'auto-évaluation générés par l'IA en fonction des recherches de l'élève.
